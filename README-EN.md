## 🔍 项目简介
智能化的鱼类表型数据采集系统，通过计算机视觉技术实现：
- 🎯 ​**14+关键体征测量**：体长、体宽、眼宽、鳍长等形态特征
- ⚖️ ​**体重同步记录**：支持电子秤数据实时采集
- 🐟 ​**多鱼种适配**：大黄鱼、加州鲈、桂花鲈等9种常见经济鱼类
- 📸 ​**智能图像处理**：复杂背景下的自动识别与测量
